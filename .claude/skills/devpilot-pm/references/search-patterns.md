# Search Pattern Templates by Industry

Templates for WebSearch queries used by the 3 research agents. Replace `{type}` with the product category.

## General (works for any product)

### Competitor Analysis
- `"{type} alternatives 2026"`
- `"{type} best tools comparison"`
- `"{type} vs {type}"`
- `"best {type} for {target_user}"`
- `"{competitor} features pricing review"`

### User Pain Points
- `"{type} pain points reddit"`
- `"{type} frustrating user experience"`
- `"{type} missing features forum"`
- `"{type} wish list feedback"`
- `"{type} complaints problems"`
- `site:reddit.com "{type}" "I wish"`
- `site:news.ycombinator.com "{type}"`

### Trends
- `"{type} trends 2026"`
- `"{type} emerging technology"`
- `"{type} market size growth"`
- `"{type} future predictions"`
- `"{related_tech} adoption enterprise"`

## SaaS / Developer Tools
- `"{type} developer experience DX"`
- `"{type} open source alternative"`
- `"{type} self-hosted"`
- `"{type} API integration"`
- `"{type} pricing too expensive"`

## Consumer / Mobile Apps
- `"{type} app store reviews negative"`
- `"{type} user retention problem"`
- `"{type} onboarding"`
- `"{type} Gen Z users"`

## E-commerce
- `"{type} conversion rate optimization"`
- `"{type} checkout abandonment"`
- `"{type} customer support automation"`

## AI / ML Products
- `"{type} AI features"`
- `"{type} LLM integration"`
- `"{type} automation workflow"`
- `"{type} AI agent"`
